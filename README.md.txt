2026 馬年日本高端伴手禮預購系統 (v4.0)這是一個專為 2026 馬年春節設計的單頁式購物預購系統 (SPA)。系統整合了精緻的 UI 設計、動態進度條、自動折扣邏輯，並透過 Google Apps Script 將訂單資料即時存儲於 Google 試算表中。系統特色高端視覺風格：採用和?子紅 (#B02A2A) 與金箔色 (#C5A059)，營造日本節慶的高級感。動態免運進度條：即時反映小計金額，引導使用者達成免運門檻（超取 $1,500 / 宅配 $3,000）。智慧折扣邏輯：於備註欄輸入隱藏版折扣碼（如：2026馬上發財 享 92 折）自動重新計算總額。完整表單處理：支援 5% 稅金計算、統編 8 碼限制、電話號碼首位補零處理（'）。後端自動化：整合 Google Sheets 存檔與 Email 自動通知管理員。技術規格前端：HTML5, Vanilla JavaScript, Tailwind CSS (CDN)。字體：Google Fonts (Noto Serif TC, Noto Sans TC)。後端：Google Apps Script (GAS)。API 模式：POST 請求 (JSON)，採用 no-cors 模式。?? 快速部署指南1. 後端設定 (Google Sheets + GAS)建立一個新的 Google 試算表。點選「擴充功能」 > 「Apps Script」。將專案中的 Code.gs 內容貼入編輯器。執行 setupSheet() 函式以初始化標題列。點選「部署」 > 「新部署」，類型選擇「網頁應用程式」，並將權限設為「所有人 (Anyone)」。複製產生的 網頁應用程式 URL。2. 前端設定 (index.html)開啟 index.html。將 GAS_URL 變數替換為您剛剛複製的網址。直接在瀏覽器開啟或上傳至網頁託管平台（如 GitHub Pages）。?? 訂單資料格式 (JSON Payload)系統送出至後端的資料結構如下：欄位說明格式範例orderId訂單編號MA7482 (MA+時間戳)name客戶姓名王小明phone聯絡電話'0912345678 (含單引號)detail商品明細山春秋-20入 x 2 \n 起司捲餅-8入 x 1discount套用折扣92折優惠 或 無total應付總額2450 (數字)isTax是否含稅是 / 否常見問題與排查 (畫面跑版解決)若您開啟 index.html 時發現畫面雜覽、跑版，請檢查以下重點：GitHub Pages 專屬排查：HTTPS 檢查：確保您的 GitHub Pages 是在 https:// 下運作。若瀏覽器攔截了非安全腳本，Tailwind CDN 會失效。強制重新整理：按下 Ctrl + F5 (Windows) 或 Cmd + Shift + R (Mac) 以清除快取並強制重新載入 CSS。檔名規範：請確保主檔案名稱為 index.html (全小寫)，GitHub Pages 對大小寫非常敏感。檢查網路連線：本系統依賴 Tailwind CSS CDN。如果您的網路環境限制了存取 cdn.tailwindcss.com，CSS 將無法加載。CDN 腳本位置：確認 HTML 檔案的 <head> 標籤內有這行：<script src="https://cdn.tailwindcss.com"></script>Viewport 設定：若在手機上顯示異常，請檢查是否有這行：<meta name="viewport" content="width=device-width, initial-scale=1.0">更新紀錄v4.0：完整收錄 17 大品牌商品、優化 Modal 成功頁面、新增 1/28 出貨提醒、修正 LINE ID 連結。v3.5：強化統編 8 碼限制邏輯、新增管理員 Email 自動通知功能。本專案由滿滿?團隊開發，僅供 2026 馬年伴手禮預購期間使用。
